# 项目说明

本项目为“菜菜论坛”后端服务，基于 Spring Boot 构建。

## 快速开始

- 运行测试：`./mvnw test`

- 本地启动（MySQL + Redis）：
  1) 复制：`docs/本地配置示例.yml` → `src/main/resources/application-local.yml`
  2) 启动：`./mvnw spring-boot:run -Dspring-boot.run.profiles=local`

## 接口与联调

- 健康检查：`GET /api/v1/health`
- 登录（JWT）：`POST /api/v1/auth/login`
- 文件上传：`POST /api/v1/files`
- 文件下载链接（预签名）：`GET /api/v1/files/presigned-get-url`
- Swagger UI：`/swagger-ui.html`

## 测试账号（本地联调）

如果你使用 `docs/sql/schema.sql` + `docs/sql/test-data.sql` 初始化本地 MySQL，可直接使用以下账号登录：

- 普通用户：`demo` / `demo`
- 管理员：`admin` / `demo`
- 禁用用户（用于验证禁用逻辑）：`disabled_user` / `demo`

## 目录结构（概览）

- `.../common`：通用横切能力（统一响应体、错误码、全局异常等）
- `.../infrastructure`：技术适配与配置（JWT、OpenAPI、MyBatis-Plus、Redis 等）
- `.../features/<功能名>`：业务功能模块
  - `domain`：领域规则
    - `port`：领域依赖的出口接口
  - `data`：端口实现（DB/Cache/外部资源）
  - `presentation`：Controller 与 DTO

更多架构说明见：`docs/架构设计.md`
