## Maven Parent 覆盖说明

Maven 的继承机制会把 parent POM 的部分元素继承到当前项目中。项目中通过“空覆盖”的方式避免继承不需要的元素（例如 `<license>`、`<developers>`）。

如果你切换了 parent 并希望继承这些元素，请移除对应的空覆盖配置。
